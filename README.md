# ESP32-2432S028 Weather Station (Enhanced Edition)

Покращена версія метеостанції на базі ESP32-2432S028 (Cheap Yellow Display).

## 📌 Засновано на роботі
Цей проект є форком та покращенням оригінального репозиторію:
- **Оригінальний автор:** marcosjl31
- **Посилання:** https://github.com/marcosjl31/WeatherStation
- **Ліцензія:** MIT

## 🚀 Власні покращення

### Апаратні
- ➕ Додано підтримку **BMP280** (температура + тиск)
- ➕ **RGB LED** з кольоровою індикацією температури
- ➕ **LDR** для автоматичної яскравості екрану

### Програмні
- ➕ Перехід з Open-Meteo на **OpenWeatherMap API**
- ➕ **Автоматичне оновлення часу** через NTP
- ➕ **Нічний режим** (21:00-07:00) з тьмяним годинником
- ➕ **Плавне згасання** яскравості
- ➕ Виправлено відображення назв днів та місяців
- ➕ Діагностика в Serial Monitor

### Функціонал
- 🌡️ Внутрішня температура (BMP280)
- 🌤️ Зовнішня погода (OpenWeatherMap)
- 💨 Атмосферний тиск
- 💧 Вологість
- 🌈 RGB індикація за температурою
- 💡 Авто-яскравість
- ⏰ Годинник з датою
- 📊 Прогноз (мін/макс)

## 🔧 Встановлення

### Бібліотеки
- TFT_eSPI
- LVGL
- ArduinoJson
- Adafruit BMP280
- WiFiManager
- ESP32Time

### Налаштування
1. Отримайте API ключ OpenWeatherMap
2. Вставте ключ в `WeatherStation.h`
3. Налаштуйте координати вашого міста
4. Завантажте код через PlatformIO
